Features

Everything a viewer should do. Nothing it shouldn't.

📄

Full GitHub-flavored rendering

Pipe tables, nested lists, task lists, fenced code blocks, blockquotes, inline images, links — rendered natively with SwiftUI.

🔄

Live auto-reload

Edit in your favorite editor, save, and the preview updates by itself — atomic saves from VS Code and friends included. ⌘R any time.

👁

Quick Look previews

Select a .md file in Finder and press Space: rendered Markdown, not plain text. No app launch needed.

⚡️

Handles huge files

A 5 MB / 115,000-block document parses in ~0.4 s in the background — the UI never freezes, scrolling stays smooth.

📋

Handy utilities

Copy the file path or the whole Markdown source with one click. Set MarkView as the default opener for .md files.

🍎

Feels like a Mac app

Because it is one. Pure SwiftPM + SwiftUI, dark & light mode, text selection, native scrolling. No web view pretending.

Why so small?

Kilobytes, not hundreds of megabytes.

Most Markdown tools ship an entire browser to draw text. MarkView renders with the frameworks already on your Mac — so the whole app is about the size of a single photo.

Privacy & security

Opening a file shouldn't be a leap of faith.

  • HTTPS-only remote imagesPlain http:// image URLs are never fetched. file:// and other schemes are always rejected.
  • Local images stay in the document folderAbsolute paths and ../ traversal that escape the opened file's directory are blocked — symlinks resolved before the check.
  • Quick Look never touches the networkPressing Space on an untrusted file fetches zero remote content; remote images show a placeholder instead.
  • Read-only by designMarkView is a viewer. It never modifies, executes, or phones home about your files.
Install

From zero to default Markdown app in 60 seconds.

Requires macOS 13+ and Swift command line tools (no full Xcode needed).

Clone the repo 
git clone https://github.com/andy51002000/MarkView.git
cd MarkView
Run the installer 
bash install.sh

Builds the app, installs to ~/Applications, registers the Quick Look extension, and offers to make MarkView the default for .md files.

Done

Double-click any .md file, or select one in Finder and hit Space.
